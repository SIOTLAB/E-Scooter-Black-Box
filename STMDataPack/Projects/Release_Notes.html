Purpose

The FP-SNS-STBOX1 The FP-SNS-STBOX1 is STM32Cube Function Pack for the Pro Mode of the:
  • SensorTile.box wireless multi sensor development kit with user-friendly app for IoT and wearable sensor applications (STEVAL-MKSBOX1V1)
  • SensorTile.box PRO discovery box with multi-sensors and wireless connectivity for any intelligent IoT node(STEVAL-MKBOXPRO)
  • STWIN.box - SensorTile Wireless Industrial Node Development Kit (STEVAL-STWINBX1)

The purpose of this functional pack is to provide simple applications and examples that show how to build custom applications for STEVAL-MKSBOX1V1 Pro Mode, STEVAL-MKBOXPRO Pro Mode and STEVAL-STWINBX1 Pro Mode.
